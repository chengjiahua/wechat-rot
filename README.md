# 微信推送机器人

非常简单的实现，只有一个依赖。
上手简单，小白也能轻松使用。

依赖：https://github.com/tans/push-bot

推送精灵的一个简单应用

定时推送消息，目前已实现消息库有彩虹屁，舔狗日志，起床语录等
